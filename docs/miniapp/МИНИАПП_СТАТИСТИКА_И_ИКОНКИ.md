# Статистика и иконки в мини-аппе (главный экран)

На главном экране отображаются три блока: **скорость**, **устройства**, **серверы**. Ниже — что сделано и какие есть ограничения.

---

## Что реализовано

### 1. Устройства (живые данные)

- **Отображение:** «X / Y устройств» — сколько устройств уже подключено по вашей ссылке (X) и лимит по тарифу (Y).
- **Источник:** API Happ (`get_install_stats`) по `install_code` из ссылки подписки. Данные приходят в `/api/miniapp/me` в полях `subscription.devices_used` и `subscription.devices_limit`.
- **Ограничение:** Работает только если с сервера доступен Happ API (при 404 от Happ показывается лимит по тарифу, без реального счётчика).

### 2. Серверы (режим реального времени)

- **Отображение:** «N серверов» или «N / M серверов» (сколько онлайн из общего числа).
- **Источник:** `GET /api/miniapp/servers-status`. Если в `.env` задан **`SERVER_LIST`** (URL через запятую), API проверяет каждый URL (HEAD/GET с таймаутом 2 с) и возвращает `online` и `total`. Мини-апп при отрисовке главного экрана запрашивает этот эндпоинт и подставляет число работающих серверов.
- **Пример .env:** `SERVER_LIST=https://server1.example.com/ping,https://server2.example.com/health`  
  Если `SERVER_LIST` не задан, используется **`SERVER_COUNT`** (одно число, без проверки) — тогда отображается просто «50 серверов».

### 3. Скорость и страна (реализовано)

- **Страна и флаг:** при загрузке главного экрана мини-апп вызывает `GET /api/miniapp/my-ip`. Бэкенд по IP запроса (X-Forwarded-For) определяет страну через ip-api.com и возвращает флаг (эмодзи) и название страны. В первом блоке отображается, например, **🇳🇱 Нидерланды** (если пользователь за VPN — это страна выхода).
- **Замер скорости:** по нажатию на первый блок (скорость) мини-апп скачивает тестовый файл 1 МБ с `GET /api/miniapp/speed-test-file`, замеряет время и показывает **XX Мбит/с**. Это скорость до вашего API-сервера (при использовании VPN трафик идёт через VPN).

## Иконки

Иконки заданы в разметке как **SVG** в `webapp/index.html` (и в `public/index.html`), блок `.stats-row`:

- Первый блок (скорость): `<svg viewBox="0 0 24 24">` с path в виде «спидометра».
- Второй блок (устройства): SVG с path «монитор».
- Третий блок (серверы): SVG с path «глобус».

**Как заменить иконку:**

1. Открыть `webapp/index.html`, найти `<div class="stats-row">` и нужный `<div class="stat-item">`.
2. Заменить внутренний `<svg>...</svg>` на другой SVG (например с [Material Icons](https://fonts.google.com/icons) или своим path) или использовать эмодзи/картинку вместо SVG.
3. Сохранить и задеплоить мини-апп (Vercel). При использовании скрипта деплоя синхронизировать с `public/index.html`, если нужно.

## Переменные и эндпоинты

| Что | Где задаётся | Эндпоинт / ответ |
|-----|----------------|------------------|
| Лимит устройств по тарифу | plan_type подписки (например 6_months_3) | В подписке уже есть, отображаем в блоке «устройства» |
| Реальный счётчик устройств (X/Y) | Happ API get_install_stats | `/api/miniapp/me` → `subscription.devices_used`, `subscription.devices_limit` |
| Количество серверов | `SERVER_COUNT` в .env | `/api/miniapp/plans` → `config.server_count` |
