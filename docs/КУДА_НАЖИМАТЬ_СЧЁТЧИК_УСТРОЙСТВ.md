# Куда нажимать: счётчик «Подключено» (пошагово)

Простая инструкция — только кнопки и экраны.

---

## Ноль. Один раз: зарегистрировать домен в Happ (на компьютере)

Чтобы счётчик «Подключено» вообще мог обновиться, домен редиректа (например `155.212.164.135`) нужно **один раз** зарегистрировать в Happ. Делается это с компьютера, где лежит проект и файл `.env`.

### По шагам (Windows)

1. **Откройте папку проекта**  
   В проводнике откройте папку: `D:\VPN BOT` (там должны быть файлы `api_miniapp.py`, `.env`, папка `scripts`).

2. **Откройте командную строку в этой папке**  
   - В проводнике в адресной строке вверху напишите `cmd` и нажмите **Enter** — откроется окно командной строки уже в папке `D:\VPN BOT`.  
   Либо: меню **Пуск** → введите **cmd** → откройте «Командная строка» → введите по очереди:
   ```text
   D:
   cd "D:\VPN BOT"
   ```

3. **Запустите скрипт**  
   В командной строке введите (можно скопировать и вставить):
   ```text
   python scripts/happ_add_redirect_domain.py
   ```
   и нажмите **Enter**.

4. **Что должно получиться**  
   - Если всё настроено: появится строка вида **`OK: домен 155.212.164.135 зарегистрирован в Happ`** или **`OK: домен ... уже зарегистрирован`** — тогда этот шаг больше делать не нужно.  
   - Если пишет **«Задайте в .env: HAPP_PROVIDER_CODE, HAPP_AUTH_KEY»** или **«Задайте в .env: HAPP_SUBSCRIPTION_REDIRECT_BASE»** — откройте файл `.env` в папке `D:\VPN BOT` и проверьте, что там есть строки:
     - `HAPP_PROVIDER_CODE=...`
     - `HAPP_AUTH_KEY=...`
     - `HAPP_SUBSCRIPTION_REDIRECT_BASE=https://155.212.164.135`  
     Сохраните `.env` и снова выполните команду из пункта 3.

После успешной регистрации домена переходите к части 1 (взять ссылку в приложении и добавить её в Happ).

---

## Часть 1. Взять ссылку для Happ (в приложении бота)

1. Откройте **Telegram** и зайдите в вашего бота (Bit VPN).
2. Внизу у бота нажмите кнопку **«Меню»** (или откройте мини-приложение из сообщения бота).
3. Откроется приложение. На **главном экране** нажмите кнопку **«Устройства»** (рядом с «Помощь»).
4. Откроется панель «Устройства». Вверху будет строка со ссылкой и кнопка **«Скопировать ссылку»** — нажмите **«Скопировать ссылку»**.
5. Ссылка скопирована в буфер. Либо нажмите **«+ Добавить устройство»** — откроется экран с той же ссылкой; там тоже есть кнопка **«Скопировать»** (иконка двух листов) — нажмите её.

Эту ссылку нужно вставить в приложение Happ (часть 2).

---

## Часть 2. Добавить ссылку в Happ (на телефоне)

1. Откройте приложение **Happ** (то, куда подключают VPN).
2. В Happ найдите раздел **«Подписки»** (или «Subscriptions») — обычно внизу или в меню.
3. Нажмите кнопку **«+»** (плюс) — «Добавить подписку».
4. Вставьте ссылку (долгим нажатием в поле ввода → «Вставить»), которую скопировали в части 1.
5. Нажмите **«Добавить»** / **«Подключить»** / **«Сохранить»** (как называется в вашей версии Happ).

Подождите 1–2 минуты.

---

## Часть 3. Обновить счётчик в приложении бота

1. Снова откройте **мини-приложение бота** (Меню бота → приложение).
2. Нажмите **«Устройства»**.
3. Рядом с надписью «Подключено» и цифрами (0/10) есть круглая кнопка **«↻»** (обновить). Нажмите **«↻»**.

После этого цифры должны обновиться (например, 1/10).

---

## Кратко

| Где | Что нажать |
|-----|------------|
| **В боте** | Меню → открыть приложение |
| **В приложении** | Кнопка **«Устройства»** |
| **В панели «Устройства»** | **«Скопировать ссылку»** или **«+ Добавить устройство»** → потом **«Скопировать»** (иконка) |
| **В Happ** | Подписки → **«+»** → вставить ссылку → Добавить |
| **Снова в приложении бота** | **«Устройства»** → кнопка **«↻»** рядом с «Подключено» |

Если после нажатия «↻» счётчик по-прежнему 0 — проверьте [СЧЁТЧИК_УСТРОЙСТВ_ПОШАГОВО.md](СЧЁТЧИК_УСТРОЙСТВ_ПОШАГОВО.md) (настройки сервера и Happ).
