# Официальный URL Happ API

По документации Happ:

- **Базовый URL API:** [https://happ-proxy.com](https://happ-proxy.com/)  
- Все эндпоинты — относительные пути к этому URL.
- Формат ответа: `application/json`.
- Результат выполнения — в полях `rc` и `msg` в теле ответа (HTTP 200).

Используемые в проекте эндпоинты:

| Метод и путь | Назначение |
|--------------|------------|
| GET `/api/add-install` | Создать лимитированную ссылку (выдача подписки пользователю). |
| GET `/api/list-install` | Список лимитированных ссылок (счётчик «Подключено» в мини-аппе). |
| GET `/api/add-domain` | Привязать домен подписки к provider (при необходимости). |

В `.env` задаётся **один** базовый URL: `HAPP_API_URL=https://happ-proxy.com`. Опционально для счётчика можно задать отдельный `HAPP_LIST_INSTALL_URL`, если у провайдера иной хост для list-install.
