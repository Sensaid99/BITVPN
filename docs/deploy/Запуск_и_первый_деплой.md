# Запуск бота и первый деплой

## 1. Библиотеки

В корне проекта один раз:

```bash
pip install -r requirements.txt
```

## 2. Настройка .env

В `.env` укажите токен бота, админов, при необходимости — способы оплаты (ЮKassa и т.д.) и `WEBAPP_URL` для кнопки мини-аппа.

## 3. Мини-апп на Vercel

- Страница мини-аппа (HTML/JS) деплоится на Vercel → получаете `https://ваш-проект.vercel.app`.
- В `.env`: `WEBAPP_URL=https://ваш-проект.vercel.app` (без слеша в конце).
- Подробно: **MINI_APP_VERCEL.md**, **API_VERCEL_ПОШАГОВО.md** в этой папке.

## 4. Запуск бота локально

```bash
python run.py
```

или `python bot/main.py`.

Демо без токена: `python demo_bot.py`.

## 5. Скрипты в корне проекта

- **ДЕПЛОЙ_НА_СЕРВЕР.bat** / **Запустить_деплой_на_сервер.vbs** — полный деплой (синхронизация, пуш, сервер, перезапуск).
- **ПУШ_НА_ГИТХАБ.bat** / **Запустить_пуш_на_гитхаб.vbs** — только пуш в GitHub (Vercel подхватит сам).
- **ЗАПУСТИТЬ_БОТА.bat**, **ЗАПУСТИТЬ_МИНИАПП_API.bat** — локальный запуск.
- Подробный список: **docs/other/КНОПКИ_И_СКРИПТЫ.txt**.
