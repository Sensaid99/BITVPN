# Запуск VPN-бота

## Быстрый старт

1. **Настройте `.env`**  
   Токен и админы уже подставлены. Для приёма оплаты укажите хотя бы один способ:
   - `YOOMONEY_TOKEN` — токен из [ЮMoney API](https://yoomoney.ru)
   - `QIWI_TOKEN` — ключ из [QIWI](https://qiwi.com)
   - `CRYPTOMUS_API_KEY` и `CRYPTOMUS_MERCHANT_ID` — для крипты через [Cryptomus](https://cryptomus.com)

2. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запуск бота**
   ```bash
   python run.py
   ```
   или
   ```bash
   python bot/main.py
   ```

## Mini App (веб-приложение для клиентов)

**Рекомендуется Vercel** — стабильный HTTPS без ngrok. Пошагово: **MINI_APP_VERCEL.md**.

- В меню бота кнопка **«📱 Открыть приложение»** показывается, только если в `.env` задан `WEBAPP_URL` (например, URL с Vercel).
- Разверните папку `webapp/` на [Vercel](https://vercel.com), скопируйте выданный `https://...` URL в `.env` → `WEBAPP_URL`.
