# Бот поддержки @HelpBit_bot

Отдельный Telegram-бот для приёма обращений пользователей: тикеты, ответы админов, уведомления, управление администраторами.

## Возможности

- **Пользователи:** создание обращения (тема + текст), мои обращения, FAQ.
- **Админы:** список открытых тикетов, ответ пользователю, закрытие тикета, статистика.
- **Главные админы (из .env):** добавление новых админов (переслать сообщение пользователя или ввести его ID).
- **Уведомления:** все админы получают уведомление о новом тикете и могут отвечать.

## Настройка

1. Скопируйте `.env.example` в `.env` в папке `support_bot`.
2. Заполните:
   - `HELPBIT_BOT_TOKEN` — токен от @BotFather для бота поддержки.
   - `HELPBIT_ADMIN_IDS` — ваш Telegram ID и других главных админов (через запятую).
   - `HELPBIT_SERVICE_NAME` — название сервиса (например BitVPN).

## Запуск

Из папки **support_bot**:

```bash
cd support_bot
python run.py
```

Или из корня проекта (VPN BOT):

```bash
python -m support_bot.run
```

Или двойной клик по **ЗАПУСТИТЬ_САППОРТ_БОТА.bat** в корне проекта (если создан).

## База данных

Файл `helpbit.db` создаётся в папке `support_bot` (тикеты, ответы, список админов). Резервные копии делайте при необходимости вручную.
